### Log
<details>
    <summary>`--log`</summary>
    `--log` can be used to set the logging level.
    <HeadlessTabs syncKey="os">
        <TabItem label="Linux">
            ~~~bash
            ./ferrumc --log=info
            ~~~
        </TabItem>
        <TabItem label="macOS">
            ~~~bash
            ./ferrumc --log=info
            ~~~
        </TabItem>
        <TabItem label="Windows">
            ~~~cmd
            .\ferrumc.exe --log=info
            ~~~
        </TabItem>
    </HeadlessTabs>
    ##### Logging values:
    - trace
    - debug
        ~~~ansi
        DEBUG ferrumc_net::server: Trying to bind to 0.0.0.0:25565
        ~~~
    - info
    - warn
    - error
</details>


<details>
    <summary>`--import`</summary>
    `--import` can be used to import a world from mca files.
    <HeadlessTabs syncKey="os">
        <TabItem label="Linux">
            ~~~bash
            ./ferrumc --import
            ~~~
        </TabItem>
        <TabItem label="macOS">
            ~~~bash
            ./ferrumc --import
            ~~~
        </TabItem>
        <TabItem label="Windows">
            ~~~cmd
            .\ferrumc.exe --import
            ~~~
        </TabItem>
    </HeadlessTabs>
    The location of these files in minecraft is explained [here](https://minecraft.wiki/w/Region_file_format#Location)
